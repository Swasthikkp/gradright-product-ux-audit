# 📖 Overview

This repository contains a comprehensive **Product UX/UI Audit** conducted on the **GradRight** mobile application as part of a Product Talent Hunt assignment.

Rather than simply identifying bugs, this project evaluates the application through the lens of a **Product Analyst**, focusing on usability, navigation, user journeys, interface consistency, accessibility, and overall product experience.

Each issue is documented with supporting evidence, reproduction steps, and actionable recommendations that can be immediately understood by Product Managers, Designers, QA Engineers, and Developers.

# 🚨 Top 3 Critical Product Findings

| Priority | Issue | Product Impact |
|-----------|-------|----------------|
| 🔴 Critical | App exits completely on swipe-back gesture from the On-Campus Programs page | Breaks expected navigation behaviour and causes immediate user drop-off |
| 🔴 Critical | Profile menu creates navigation dead-ends on the Community page | Prevents users from continuing their journey and creates frustrating UX |
| 🔴 Critical | Community lacks moderation and self-management features | Introduces safety, trust, and platform credibility risks |
